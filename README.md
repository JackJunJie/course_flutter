## 实现内容
* mvp模式
* 使用provider 做状态管理
* 下拉刷新 + 上拉加载更多
* 模块化路由管理
* 使用Sliver 系列组件实现复杂滚动效果
## 注意事项
* 保利威视视频播放目前仅支持Android
